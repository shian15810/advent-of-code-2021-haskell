# advent-of-code-2021-haskell

<https://adventofcode.com/2021>

```man
USAGE
        aoc DAY [INPUT]

DAY
        In the format of 'day-DD', e.g. 'day-10'.

INPUT
        Only 'example' and 'puzzle' are allowed.
        Default to 'puzzle' if unspecified.

EXAMPLES
        aoc day-01
        aoc day-02 example
        aoc day-03 puzzle
```
